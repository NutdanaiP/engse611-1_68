เกี่ยวกับฉัน

สวัสดีครับผมชื่อ ณัฐดนัย แปงจิตต์ เป็นนักศึกษาวิศวกรรมซอฟต์แวร์

ผมสนใจในการเขียนโปรแกรมและต้องการเป็นนักพัฒนาเว็บไซต์ในอนาคต

การศึกษา

ปัจจุบัน

  • มหาวิทยาลัย: มหาวิทยาลัยเทคโนโลยีราชมงคลล้านนา เชียงใหม่
  • คณะ: วิศวกรรมศาสตร์
  • สาขา: วิศวกรรมซอฟต์แวร์
  • ชั้นปีที่: 1

งานอดิเรกและความสนใจ

สิ่งที่ฉันชอบทำ:

  1. เล่นเกมคอมพิวเตอร์
  2. ดูหนัง
  3. ฟังเพลง
  4. อ่านหนังสือเทคโนโลยี

ทักษะที่กำลังเรียนรู้:

  • HTML & CSS (ยังเริ่มต้น)
  • JavaScript (ยังเริ่มต้น)
  • Python (ยังเริ่มต้น)

เป้าหมายในอนาคต

ผมมีความตั้งใจที่จะพัฒนาทักษะการเขียนโค้ดให้ดียิ่งขึ้น เป้าหมายในอนาคตของผมคือการเป็นนักพัฒนาซอฟต์แวร์ และสามารถสร้างสรรค์ผลงานที่เป็นประโยชน์และมีคุณภาพสูง ผมเชื่อว่าการเรียนรู้ไม่สิ้นสุด และพร้อมที่จะเรียนรู้เทคโนโลยีใหม่ๆ เพื่อบรรลุเป้าหมายในอนาคต
